#  Smart Workspace Manager

##  What is this project?

Smart Workspace Manager is a web-based productivity system that helps users organize their work in a structured way.  
Users can create workspaces, manage projects, store documents, upload images, and track everything in one place.

It is designed as a **simple SaaS-style productivity platform** for students and professionals.

---

##  Features

- User registration and login (JWT authentication)
- Create multiple workspaces
- Create and manage projects inside workspaces
- Add and manage documents/notes
- Upload and manage images
- Dashboard to view everything in one place
- Search functionality for quick access

---
