# User Stories

## Authentication

### US-01
As a user, I want to register an account so that I can access the platform.

### US-02
As a user, I want to log in securely so that I can access my workspaces.

---

## Workspace Management

### US-03
As a user, I want to create a workspace so that I can organize my projects.

### US-04
As a user, I want to edit workspace information so that I can keep it updated.

### US-05
As a user, I want to delete a workspace so that I can remove unused workspaces.

---

## Project Management

### US-06
As a user, I want to create projects inside a workspace so that I can organize my work.

### US-07
As a user, I want to update project information so that project details remain accurate.

### US-08
As a user, I want to delete projects so that I can remove unnecessary data.

---

## Document & Notes Management

### US-09
As a user, I want to create notes so that I can store important information.

### US-10
As a user, I want to edit notes so that I can keep information up to date.

### US-11
As a user, I want to delete notes so that I can remove outdated information.

---

## Image Management

### US-12
As a user, I want to upload images so that I can store project-related visuals.

### US-13
As a user, I want to view uploaded images so that I can access visual resources.

### US-14
As a user, I want to delete images so that I can remove unnecessary files.

---

## Search Functionality

### US-15
As a user, I want to search documents and images so that I can quickly find information.

---

## Dashboard

### US-16
As a user, I want to view a dashboard so that I can monitor workspaces, projects, and files efficiently.
